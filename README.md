# Bloom & Petal — Магазин цветов с предзаказом на Django 

Полноценный веб-сайт на Django для магазина цветов с системой предзаказов.

##  Инструкция запуска проекта

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Применение миграций

```bash
python manage.py migrate
```

### 3. Создание суперпользователя (администратора)

```bash
python manage.py createsuperuser
```

Введите имя пользователя, email и пароль.

### 4. Запуск сервера разработки

```bash
python manage.py runserver
```

Сайт будет доступен по адресу: http://127.0.0.1:8000/

Админ-панель: http://127.0.0.1:8000/admin/

##  Функциональность

### Для клиентов:
- ✅ Просмотр каталога букетов
- ✅ Детальная страница товара с выбором размера
- ✅ Корзина с добавлением/удалением товаров
- ✅ Изменение количества товаров в корзине
- ✅ Оформление предзаказа с выбором даты доставки
- ✅ Автоматический расчёт доставки (бесплатно от 3000 ₽)
- ✅ Сохранение корзины в сессии (работает для анонимных пользователей)

### Для администратора:
- ✅ Управление товарами через админ-панель Django
- ✅ Добавление вариантов размеров (S, M, L) с разными ценами
- ✅ Управление доступностью вариантов (флаг `is_available`)
- ✅ Просмотр и управление заказами
- ✅ Автоматический расчёт суммы заказа

##  Модели данных

### Product (Товар)
- `name` — Название букета
- `description` — Описание
- `image` — Изображение
- `is_active` — Активен ли товар
- `created_at`, `updated_at` — Даты создания/обновления

### ProductVariant (Вариант товара)
- `product` — Связь с товаром (ForeignKey)
- `size` — Размер (S, M, L)
- `price` — Цена
- `is_available` — Доступен для предзаказа

### Order (Заказ)
- `first_name`, `phone`, `email` — Данные клиента
- `address` — Адрес доставки
- `delivery_date` — Дата доставки
- `total_cost`, `delivery_cost` — Стоимость
- `status` — Статус заказа (новый, подтверждён, в работе, выполнен, отменён)

##  Логика корзины

Корзина реализована на Django Sessions (`apps/catalog/cart.py`):
- Хранит ID вариантов товаров и их количество
- Автоматически подтягивает актуальные данные из БД при отображении
- Рассчитывает стоимость доставки (бесплатно от 3000 ₽)
- Работает для анонимных пользователей

##  Настройки (config/settings.py)

```python
CART_SESSION_ID = 'cart'                    # Ключ сессии для корзины
FREE_DELIVERY_THRESHOLD = 3000              # Порог бесплатной доставки
STANDARD_DELIVERY_COST = 500                # Стоимость обычной доставки
```

##  Добавление тестовых данных

Для добавления тестовых товаров:

1. Зайдите в админ-панель: http://127.0.0.1:8000/admin/
2. Войдите под суперпользователем
3. Перейдите в раздел "Товары"
4. Нажмите "Добавить товар"
5. Заполните название, описание, загрузите изображение
6. В разделе "Варианты товаров" добавьте размеры (S, M, L) с ценами
